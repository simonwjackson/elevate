Moonbeam is a simple, automated, context-aware launcher for [Moonlight](https://moonlight-stream.org).

:::tip[Ideal for Linux Nomads]
Moonbeam is ideal for portable Linux devices like the [Steam Deck](https://store.steampowered.com/steamdeck) who often remote into another computer outside of the local network
:::

## Moonlight Comparison

Moonbeam builds upon Moonlight by adding some missing features. Moonbeam only extends the functionality of moonlight and therefore is fully backwards compatible.

| Feature | Moonbeam | Standard Moonlight |
|---------|----------|---------------------|
| Pre-stream settings optimization | <Badge text="Yes" variant="success"/> | <Badge text="No" variant="danger"/> |
| Fast fail on poor connections  | <Badge text="Yes" variant="success"/> | <Badge text="No" variant="danger"/> |
| Prioritize FPS, resolution, or bandwidth | <Badge text="Yes" variant="success"/> | <Badge text="No" variant="danger"/> |
| Auto-reconnect functionality | <Badge text="Yes" variant="success"/> | <Badge text="No" variant="danger"/> |
| Integrate with other game launchers | <Badge text="Yes" variant="success"/> | <Badge text="Yes" variant="success"/> 

:::note
Moonbeam is not a standalone streaming solution. It works in conjunction with Moonlight and requires Sunshine host on your gaming PC.
:::
